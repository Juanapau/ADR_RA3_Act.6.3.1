Diapositiva 1

¿Qué es un Diagrama de Flujo?

Es una representación gráfica de los pasos de un proceso, usando símbolos conectados con flechas que muestran el orden y las decisiones a tomar.
💡 Piénsalo así:
Igual que Google Maps te indica cada paso para llegar a un destino, el diagrama le dice al sistema qué hacer en cada momento.

Diapositiva 3

El proceso de acceso a un reporte

🔐 1. AUTENTICACIÓN
El usuario ingresa su usuario y contraseña. El sistema verifica si son correctos.
📋 2. SELECCIÓN
El sistema muestra el menú. El usuario selecciona el módulo de reportes.
⚙️ 3. GENERACIÓN
El sistema busca y procesa los datos para generar el reporte solicitado.
📤 4. EXPORTACIÓN
El usuario descarga el reporte en PDF o Excel para usarlo.

20–40 min 👥 ¡Manos a la obra! — Diseña tu diagrama en pareja ▼

Ahora es tu turno. Junto a tu pareja, diseñarás un diagrama de flujo completo del proceso de acceso a un reporte empresarial. Descarga la plantilla en PowerPoint y completa el siguiente ejercicio:

ENUNCIADO TechReport Solutions

La empresa TechReport Solutions necesita automatizar el acceso a sus reportes financieros. Actualmente, cuando un empleado quiere consultar un reporte, el proceso se realiza de forma manual y sin control. La empresa quiere representar gráficamente el nuevo proceso digital.

Diseña un diagrama de flujo que muestre el proceso completo de acceso a un reporte en el sistema, considerando los siguientes pasos:

  1. El empleado ingresa al sistema con su usuario y contraseña.
  2. El sistema verifica si las credenciales son correctas. Si no lo son, permite reintentar. Si son correctas, continúa.
  3. El empleado selecciona el módulo de Reportes en el menú principal.
  4. El sistema genera el reporte solicitado.
  5. El empleado exporta el reporte en formato PDF o Excel.
  6. El proceso finaliza.
Usa la simbología ISO estándar, aplica los 5 símbolos estudiados y entrega el archivo con el nombre:
Diagrama_[TuNombre]_[NombrePareja]
